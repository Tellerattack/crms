利用springboot+mybatis打造一个简易的客户管理系统
===
## 系统结构图

![系统结构图](https://github.com/bestrow/crms/blob/master/other-img/crms.jpg)

## 开发环境

* **操作系统**：Windows 10
* **IDE工具**：IDEA
* **JDK**：JDK1.8
* **数据库**：MySQL 5.7
* **构建工具**：Maven
* **框架**：Springboot+Mybatis

## 相关技术介绍

>MyBatis

* DAO层设计与开发
* MyBatis作为持久层
* 与Springboot整合

>Springboot

* 整合Bean、Service、Controller
* Restful借口设计和使用

>前端

* Thymeleaf
* Jquery（ajax传值）
* HTML
